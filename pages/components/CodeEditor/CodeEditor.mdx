# Editor
The Code Editor component allows users to write, edit, and view code within an application. It's used when:
- Users need to create or edit large blocks of code
- Displaying large code snippets.

## Anatomy
Key anatomical elements of a code editor:
- Container
- Query code editor area: The main area where code is written and displayed.
- Line numbers column: Displayed in the left margin of the code window. 
- Actions area like Copy.
- Label.
- Help action.
- Helper text/ Validation text.
