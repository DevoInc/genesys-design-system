# Icon
The Icon component is an essential element in design systems, used to visually represent actions, categories, or content in a minimal and recognizable form. Icons enhance the user interface by providing quick, at-a-glance recognition of functions, improving both usability and aesthetics.

## Sizes

Icons should be scaled according to the size of the text. Sizes: xxxxs, xxxs, xxs, xs, sm, md, lg, xl, xxl, xxxl, xxxxl.

## Color

The colour of the icon has predefined colours and should be consistent with the colour of its surroundings, unless the icon is being used to express state and it should be coloured accordingly.
