# Inline message

The Inline Message component gives additional information to users about an action requested to complete the current task and displays confirmations, errors, or warnings related to a component.

<section className="row gy-4 row-cols-1 row-cols-sm-2 margin-top--lg">
  <div class="col-8">
<p class="sbdocs sbdocs-p margin-none">To reveal the message, users have to click or hover on a trigger item like an Action, a Link, or a Label Component. The Inline Message then could be appear attached to the trigger or floating over the rest of the page elements in the upper right corner. In this case, the Inline Message is also draggable and it could be moved to any place on the page.</p>
